## WantedCardHorizontal

`WantedCardHorizontal`은 썸네일과 텍스트 설명이 나란히 배치되는 가로형 카드 컴포넌트입니다.  
Skeleton UI를 통해 로딩 상태를 표현할 수 있으며, 좌우/상하 커스텀 콘텐츠 영역도 제공합니다.  
리스트 형태의 UI, 선택형 항목 등에 유용하게 사용됩니다.

### 사용 예시
```kotlin
WantedCardHorizontal(
    modifier = Modifier.fillMaxWidth(),
    title = "제목",
    caption = "설명 텍스트",
    extraCaption = "추가 설명",
    topContent = {
        WantedContentBadge(text = "상단")
    },
    bottomContent = {
        WantedContentBadge(text = "하단")
    },
    leftContent = {
        WantedCheckBox(
            modifier = Modifier,
            size = CheckBoxSize.Normal,
            onCheckedChange = {}
        )
    },
    rightContent = {
        Icon(
            painter = painterResource(id = R.drawable.icon_bookmark_fill_svg),
            contentDescription = ""
        )
    },
    onClick = { /* 클릭 처리 */ }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| modifier | `Modifier` | 카드의 외형 및 배치를 설정합니다. |
| thumbnail | `Any?` | Glide를 사용해 렌더링할 이미지 소스입니다. URL 또는 리소스 객체가 가능합니다. |
| title | `String` | 카드의 제목입니다. |
| caption | `String` | 제목 하단의 보조 설명입니다. |
| extraCaption | `String` | 추가 설명 텍스트입니다. |
| isLoading | `Boolean` | true일 경우 스켈레톤 UI로 대체 렌더링됩니다. |
| cardDefault | `WantedCardDefault` | Skeleton 렌더링 시 항목별 표시 여부를 조절할 수 있는 설정 객체입니다. |
| topContent | `@Composable (() -> Unit)?` | 설명 영역 상단에 표시될 커스텀 콘텐츠입니다. |
| bottomContent | `@Composable (() -> Unit)?` | 설명 영역 하단에 표시될 커스텀 콘텐츠입니다. |
| leftContent | `@Composable (() -> Unit)?` | 썸네일 왼쪽에 표시될 콘텐츠입니다. (예: 체크박스 등) |
| rightContent | `@Composable (() -> Unit)?` | 설명 오른쪽에 표시될 콘텐츠입니다. (예: 북마크 아이콘 등) |
| onClick | `() -> Unit` | 카드 클릭 시 호출되는 콜백입니다. |

## WantedCardDefault

스켈레톤 렌더링 중 어떤 항목을 표시할지를 설정하는 데이터 클래스입니다.  
로딩 상태에서 UI를 세부적으로 제어할 수 있도록 도와줍니다.

### 사용 예시
```kotlin
val config = WantedCardDefaults.getDefault(
    topContentSkeleton = true,
    bottomContentSkeleton = true
)
```

### 프로퍼티

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| topContentSkeleton | `Boolean` | 상단 커스텀 콘텐츠 영역의 스켈레톤 표시 여부 |
| captionSkeleton | `Boolean` | 기본 캡션에 대한 스켈레톤 표시 여부 |
| extraCaptionSkeleton | `Boolean` | 추가 캡션 스켈레톤 표시 여부 |
| bottomContentSkeleton | `Boolean` | 하단 콘텐츠 스켈레톤 표시 여부 |

---



## Note
- `thumbnail`이 null일 경우 회색 배경 Placeholder로 대체됩니다.
- `leftContent`, `rightContent`를 통해 유연한 사용자 정의가 가능합니다.
- `isLoading`과 `cardDefault` 설정을 조합해 다양한 스켈레톤 UI를 연출할 수 있습니다.
