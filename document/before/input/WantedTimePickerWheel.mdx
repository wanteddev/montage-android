## WantedTimePickerWheel

`WantedTimePickerWheel`은 사용자가 오전/오후, 시(hour), 분(minute)을 각각 휠 형태로 선택할 수 있는 모달 컴포넌트입니다.  
휠 UI는 `WantedNumberPicker`, `WantedStringPicker`로 구성되며, 선택된 값은 콜백을 통해 전달됩니다.  
선택 가능 범위는 `WantedTimePickerWheelDefault` 객체를 통해 설정할 수 있습니다.

### 사용 예시
```kotlin
WantedTimePickerWheel(
    isAm = true,
    hour = 10,
    minute = 30,
    confirm = "확인",
    cancel = "취소",
    title = "시간 선택",
    onSelected = { isAm, hour, minute ->
        // 시간 처리
    },
    onDismissRequest = { /* 다이얼로그 닫힘 처리 */ }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| isAm | `Boolean` | 초기 오전/오후 여부입니다. true는 AM, false는 PM입니다. |
| hour | `Int` | 초기 선택된 시(hour)입니다. |
| minute | `Int` | 초기 선택된 분(minute)입니다. |
| confirm | `String` | 확인 버튼에 표시할 텍스트입니다. |
| cancel | `String` | 취소 버튼에 표시할 텍스트입니다. |
| modifier | `Modifier` | 외형을 설정하는 Modifier입니다. |
| title | `String` | 다이얼로그 상단에 표시될 제목입니다. |
| default | `WantedTimePickerWheelDefault` | 시간 선택 휠 설정 객체입니다. |
| onSelected | `(Boolean, Int, Int) -> Unit` | 선택된 AM 여부, 시, 분을 전달하는 콜백입니다. |
| onDismissRequest | `() -> Unit` | 다이얼로그 닫기 요청 시 호출되는 콜백입니다. |

---



## WantedTimePickerWheelDefaults.getDefault

휠 형태의 시간 선택에 사용할 수 있는 기본 설정 값을 반환합니다.  
시간/분의 허용 범위, 오전/오후 선택 가능 여부, 비활성 시간 숨김 여부 등을 세부 설정할 수 있습니다.

### 사용 예시
```kotlin
val default = WantedTimePickerWheelDefaults.getDefault(
    enableAm = true,
    enablePm = true,
    enableMinHour = 9,
    enableMaxHour = 18,
    enableMinMinute = 0,
    enableMaxMinute = 45,
    isHideDisableTime = true
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| enableAm | `Boolean` | 오전(AM) 선택을 허용할지 여부입니다. 기본값은 true입니다. |
| enablePm | `Boolean` | 오후(PM) 선택을 허용할지 여부입니다. 기본값은 true입니다. AM이 false인 경우 자동으로 true로 설정됩니다. |
| enableMinHour | `Int` | 선택 가능한 최소 시(hour)입니다. 1 이상이어야 하며, 기본값은 1입니다. |
| enableMaxHour | `Int` | 선택 가능한 최대 시(hour)입니다. `enableMinHour`보다 작으면 자동 보정됩니다. 기본값은 12입니다. |
| enableMinMinute | `Int` | 선택 가능한 최소 분(minute)입니다. 기본값은 0입니다. |
| enableMaxMinute | `Int` | 선택 가능한 최대 분(minute)입니다. `enableMinMinute`보다 작으면 자동 보정됩니다. 기본값은 59입니다. |
| isHideDisableTime | `Boolean` | 비활성 시간대를 숨길지 여부입니다. 기본값은 false입니다. |

---



## Note
- 오전/오후(AM/PM), 시(hour), 분(minute)을 개별 휠로 선택 가능하며, 유효성 검증과 활성화 여부에 따라 확인 버튼이 제어됩니다.
- 설정 객체를 통해 시간 제한, 숨김 조건 등 다양한 커스터마이징이 가능합니다.
