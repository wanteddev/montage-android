## WantedTextField

`WantedTextField`는 단일 줄 또는 다중 줄 입력을 위한 커스텀 텍스트 입력 컴포넌트입니다. `String` 또는 `TextFieldValue`를 통해 상태를 제어할 수 있으며, 상태 메시지, 버튼, 아이콘 등의 다양한 기능을 제공합니다.

### 사용 예시
```kotlin
WantedTextField(
    text = "입력된 값",
    placeholder = "텍스트를 입력하세요",
    title = "이름",
    description = "2자 이상 입력해 주세요",
    rightButton = "완료",
    onValueChange = { value -> /* 처리 */ }
)
```

### 파라미터

### WantedTextField (String 기반)
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| text | `String` | 현재 입력된 텍스트입니다. |
| modifier | `Modifier` | 외형 및 배치 설정입니다. |
| placeholder | `String` | 입력 필드에 표시할 플레이스홀더입니다. |
| title | `String` | 상단 제목 텍스트입니다. |
| description | `String?` | 하단 설명 또는 상태 메시지입니다. |
| rightButton | `String?` | 입력 우측에 표시할 버튼 텍스트입니다. |
| rightButtonVariant | `RightVariant` | 우측 버튼 스타일입니다. 기본값은 Normal입니다. |
| status | `Status` | 입력 필드 상태 (Normal, Positive, Negative)입니다. |
| enabled | `Boolean` | 입력 가능 여부입니다. 기본값은 true입니다. |
| rightButtonEnabled | `Boolean` | 우측 버튼의 활성화 여부입니다. |
| maxLines | `Int` | 최대 줄 수입니다. 기본값은 1입니다. |
| minLines | `Int` | 최소 줄 수입니다. 기본값은 1입니다. |
| maxWordCount | `Int` | 최대 글자 수입니다. 기본값은 2000자입니다. |
| enabledOverflowText | `Boolean` | 글자 수 초과 허용 여부입니다. 기본값은 false입니다. |
| requiredBadge | `Boolean` | 필수 입력 표시 뱃지 여부입니다. |
| interactionSource | `MutableInteractionSource` | 포커스 추적용 상태 객체입니다. |
| focused | `State<Boolean>` | 외부에서 포커스 여부를 제어할 수 있습니다. |
| keyboardOptions | `KeyboardOptions` | 키보드 옵션입니다. |
| keyboardActions | `KeyboardActions` | 키보드 액션 처리 설정입니다. |
| background | `Color` | 배경 색상입니다. |
| leadingIcon | `(() -> Unit)?` | 좌측 아이콘 슬롯입니다. |
| trailingIcon | `(() -> Unit)?` | 우측 아이콘 슬롯입니다. |
| rightContent | `((Dp) -> Unit)?` | 우측 콘텐츠 슬롯입니다. |
| onClickRightButton | `() -> Unit` | 우측 버튼 클릭 시 동작입니다. |
| onValueChange | `(String) -> Unit` | 입력 값이 변경될 때 호출됩니다. |

## Enum 설명

### Status
| 값 | 설명 |
|:---|:---|
| Normal | 기본 상태입니다. |
| Positive | 성공/완료 상태입니다. |
| Negative | 에러/실패 상태입니다. |

### RightVariant
| 값 | 설명 |
|:---|:---|
| Normal | 강조된 버튼 스타일입니다. |
| Assistive | 보조적이고 약한 버튼 스타일입니다. |

---



## Note
- `TextFieldValue`를 사용할 경우 커서 및 선택 영역 제어가 가능합니다.
- `enabledOverflowText`가 false인 경우, `maxWordCount` 이상 입력 시 제한됩니다.
- 다양한 아이콘, 뱃지, 버튼 슬롯을 통해 유연한 확장이 가능합니다.
