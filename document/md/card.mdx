## WantedCardDefaults
카드 UI의 스켈레톤 표시 여부를 지정하는 데이터 클래스입니다. 각 항목별로 스켈레톤을 표시할지 여부를 설정합니다.

## 사용 예시
```kotlin
val config = WantedCardDefaults.getDefault(
    topContentSkeleton = true,
    bottomContentSkeleton = true
)
```
### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| topContentSkeleton | `Boolean` | 상단 콘텐츠 영역에 스켈레톤을 표시할지 여부입니다. |
| captionSkeleton | `Boolean` | 메인 캡션에 스켈레톤을 표시할지 여부입니다. |
| extraCaptionSkeleton | `Boolean` | 추가 캡션에 스켈레톤을 표시할지 여부입니다. |
| bottomContentSkeleton | `Boolean` | 하단 콘텐츠 영역에 스켈레톤을 표시할지 여부입니다. |

## WantedCardHorizontal (가로형 카드 컴포저블입니다.)
썸네일과 설명 영역을 좌우로 배치한 형태의 카드입니다. 로딩 상태에서는 Skeleton UI가 나타나며, 상단/하단 커스텀 콘텐츠와 좌우 콘텐츠 Slot을 제공합니다.

### 사용 예시
```kotlin
WantedCardHorizontal(
    modifier = Modifier.fillMaxWidth(),
    title = "제목",
    caption = "캡션",
    extraCaption = "추가 설명",
    topContent = { WantedContentBadge(text = "상단") },
    bottomContent = { WantedContentBadge(text = "하단") },
    leftContent = { WantedCheckBox(...) },
    rightContent = { Icon(...) },
    onClick = { /* 클릭 처리 */ }
)
```
### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| modifier | Modifier | 컴포넌트의 레이아웃과 스타일을 조정합니다. |
| thumbnail | `Any?` | 썸네일 이미지 리소스 또는 URL입니다. null이면 표시되지 않습니다. |
| title | `String` | 카드의 주요 제목입니다. |
| caption | `String` | 제목 아래에 표시될 보조 설명입니다. |
| extraCaption | `String` | 추가적인 설명 텍스트입니다. |
| isLoading | `Boolean` | true일 경우 스켈레톤 UI로 렌더링됩니다. |
| cardDefault | `WantedCardDefault` | 스켈레톤 모드 시 항목별 표시 여부를 지정하는 설정 객체입니다. |
| topContent | `@Composable (() -> Unit)?` | 설명 위에 표시될 추가 콘텐츠입니다. |
| bottomContent | `@Composable (() -> Unit)?` | 설명 아래에 표시될 추가 콘텐츠입니다. |
| leftContent | `@Composable (() -> Unit)?` | 썸네일 왼쪽에 표시될 콘텐츠입니다. (예: 체크박스) |
| rightContent | `@Composable (() -> Unit)?` | 설명 오른쪽에 표시될 콘텐츠입니다. (예: 아이콘 버튼) |
| onClick | `() -> Unit` | 카드 클릭 시 호출되는 콜백 함수입니다. |

## WantedCardVertical (세로형 카드 컴포저블입니다.)
로딩 상태(`isLoading`)에 따라 Skeleton 또는 실제 콘텐츠를 렌더링하며, 썸네일, 오버레이 캡션, 타이틀, 캡션, 서브캡션, 추가 캡션, 상/하단 커스텀 콘텐츠를 지원합니다.

### 사용 예시
```kotlin
WantedCardVertical(
    title = "제목",
    caption = "캡션",
    overlayCaption = "Overlay",
    overlayToggleIcon = {
        Icon(painter = painterResource(id = R.drawable.icon), contentDescription = null)
    },
    onClick = { /* 클릭 이벤트 */ }
)
```
### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| modifier | Modifier | 레이아웃 및 스타일을 지정하는 Modifier입니다. |
| thumbnail | `Any?` | 썸네일 이미지 리소스 또는 URL입니다. null일 경우 표시되지 않습니다. |
| overlayCaption | `String` | 썸네일 위에 오버레이로 표시할 텍스트입니다. |
| title | `String` | 카드 타이틀 텍스트입니다. |
| caption | `String` | 보조 캡션 텍스트입니다. |
| subCaption | `String` | 추가 보조 캡션 텍스트입니다. |
| extraCaption | `String` | 하단 추가 설명 텍스트입니다. |
| isLoading | `Boolean` | 로딩 상태 여부입니다. true이면 skeleton UI가 렌더링됩니다. |
| cardDefault | `WantedCardDefault` | skeleton 모드에서 사용할 설정값입니다. |
| overlayToggleIcon | `@Composable (() -> Unit)?` | 썸네일 오버레이에 포함될 토글 아이콘입니다. |
| topContent | `@Composable (() -> Unit)?` | 카드 상단 타이틀 위에 추가 표시할 컴포저블입니다. |
| bottomContent | `@Composable (() -> Unit)?` | 카드 하단에 추가 표시할 컴포저블입니다. |
| onClick | `() -> Unit` | 카드 전체 클릭 시 호출되는 콜백입니다. |

---
