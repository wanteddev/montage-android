## WantedDotIndicator
페이지네이션용 dot indicator를 표시하는 컴포저블입니다. 현재 페이지 위치를 중심으로 일정 개수의 dot을 보여주며, 선택 상태에 따라 색상, 크기, 애니메이션이 다르게 표시됩니다.

## 사용 예시
```kotlin
WantedDotIndicator(
    totalPageCount = 10,
    visibleDotCount = 5,
    currentIndex = 2
)
```

### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| totalPageCount | `Int` | 전체 페이지 수입니다. |
| visibleDotCount | `Int` | 화면에 표시할 최대 dot 수입니다. |
| currentIndex | `Int` | 현재 선택된 페이지 index입니다. |
| modifier | `Modifier` | 배치와 외형을 위한 Modifier입니다. |
| size | `WantedDotIndicatorSize` | dot의 크기를 결정합니다. 기본값은 Medium입니다. |
| type | `WantedDotIndicatorType` | dot 스타일 타입입니다. 기본값은 Normal입니다. |

---

## WantedPageCounter
현재 페이지와 전체 페이지 수를 함께 표시하는 페이지 카운터 컴포저블입니다. `2 / 10` 형식으로 페이지 정보를 출력하며, 일반형과 alternative 스타일(투명 배경) 중 선택할 수 있습니다.

## 사용 예시
```kotlin
WantedPageCounter(
    totalPageCount = 10,
    currentIndex = 2,
    isAlternative = true
)
```

### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| totalPageCount | `Int` | 전체 페이지 수입니다. |
| currentIndex | `Int` | 현재 페이지 인덱스입니다. |
| modifier | `Modifier` | 배치 및 외형 설정을 위한 Modifier입니다. |
| size | `WantedPageCounterSize` | 크기 설정입니다 (Small 또는 Normal). |
| isAlternative | `Boolean` | alternative 배경 스타일 적용 여부입니다. |

---

## Enum 설명
### WantedDotIndicatorSize
| 값 | 설명 |
|:---|:---|
| Small | 2~6dp 범위의 작은 크기 |
| Medium | 6~10dp 범위의 중간 크기 |

### WantedDotIndicatorType
| 값 | 설명 |
|:---|:---|
| Normal | 배경이 채워진 원형 스타일 |
| White | 흰색 테두리와 반투명 배경 스타일 |

### WantedPageCounterSize
| 값 | 설명 |
|:---|:---|
| Small | 좌우 10dp, 상하 4dp, 간격 3dp |
| Normal | 좌우 12dp, 상하 6dp, 간격 4dp |

---

## Note
object WantedPaginationContract는 페이지네이션 컴포넌트에서 사용되는 설정 값을 정의하는 객체입니다. Dot indicator, Page counter와 관련된 시각적 스타일과 크기를 제어할 수 있는 enum 클래스를 포함합니다.
