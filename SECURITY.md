# Security Policy

## Supported Versions

The following versions of Montage Android are currently supported with security updates:

| Version | Supported          |
|---------| ------------------ |
| ^3.1.0  | :white_check_mark: |
| < 3.0   | :x:                |

---

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue in Montage Android, please report it responsibly.

### How to Report

Please report security vulnerabilities through the **GitHub Security tab** → **Advisories** → **Report a vulnerability**.


### What to Include

When reporting a vulnerability, please include:

* **Description**: A clear description of the vulnerability
* **Impact**: What could be exploited and the potential impact
* **Reproduction Steps**: Detailed steps to reproduce the issue
* **Environment**: Android API level, library version, device information
* **Proof of Concept**: If applicable, provide a minimal code example
* **Suggested Fix**: If you have suggestions for fixing the issue
