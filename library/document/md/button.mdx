## WantedButton
다양한 스타일의 버튼을 생성하는 공통 Compose 함수입니다. ButtonVariant에 따라 Solid, Outlined, Text 버튼을 선택하여 렌더링합니다.

```kotlin
WantedButton(
    text = "확인",
    variant = ButtonVariant.SOLID,
    type = ButtonType.PRIMARY,
    size = ButtonSize.LARGE,
    onClick = { /* 클릭 이벤트 처리 */ }
)
```

### Parameters
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| text | String | 버튼에 표시할 텍스트입니다. |
| modifier | Modifier | Modifier를 통해 버튼 외형을 조정합니다. |
| type | ButtonType | 버튼의 타입(PRIMARY, SECONDARY, ASSISTIVE)을 지정합니다. |
| size | ButtonSize | 버튼의 크기(LARGE, MEDIUM, SMALL)를 지정합니다. |
| variant | ButtonVariant | 버튼의 형태(SOLID, OUTLINED, TEXT)를 지정합니다. |
| enabled | Boolean | 버튼의 활성화 여부를 지정합니다. |
| isLoading | Boolean | 로딩 상태를 표시할지 여부입니다. |
| leadingDrawable | Int? | 버튼 왼쪽에 표시할 Drawable 리소스 ID입니다. |
| trailingDrawable | Int? | 버튼 오른쪽에 표시할 Drawable 리소스 ID입니다. |
| onClick | () -> Unit | 버튼 클릭 시 호출되는 콜백입니다. |

## Overload
WantedButton을 더 세밀하게 제어하기 위한 Compose 함수입니다. WantedButtonDefault를 직접 주입하여 스타일과 상태를 설정할 수 있습니다.

```kotlin
WantedButton(
    text = "삭제",
    buttonDefault = WantedButtonDefaults.getDefault(ButtonVariant.OUTLINED),
    onClick = { /* 클릭 처리 */ }
)
```
