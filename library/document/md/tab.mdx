## WantedActionArea
하단에 위치한 액션 버튼 영역을 생성합니다. positive, negative, neutral 텍스트를 통해 버튼을 생성하며, 각 클릭 콜백도 함께 전달합니다. variant 영역을 통해 상단에 부가 요소를 렌더링할 수 있습니다.

## 사용 예시
```kotlin
WantedActionArea(
    type = ActionAreaType.Strong,
    positive = "확인",
    onClickPositive = { /* 처리 */ },
    negative = "취소",
    onClickNegative = { /* 처리 */ },
    neutral = "건너뛰기",
    onClickNeutral = { /* 처리 */ }
)
```

### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| type | `ActionAreaType` | 액션 영역의 타입을 설정합니다. |
| positive | `String` | 메인(긍정) 액션 버튼 텍스트입니다. |
| onClickPositive | `() -> Unit` | 메인 액션 버튼 클릭 콜백입니다. |
| negative | `String` | 서브(부정) 액션 버튼 텍스트입니다. |
| onClickNegative | `() -> Unit` | 서브 액션 버튼 클릭 콜백입니다. |
| neutral | `String` | 추가(중립) 액션 버튼 텍스트입니다. |
| onClickNeutral | `() -> Unit` | 추가 액션 버튼 클릭 콜백입니다. |
| caption | `String?` | 액션 영역 상단에 표시할 캡션입니다. |
| scrollableState | `ScrollableState?` | 스크롤 가능한 경우 상태를 전달합니다. |
| modifier | `Modifier` | Modifier를 설정합니다. |
| background | `Boolean` | 배경 그라데이션 표시 여부를 지정합니다. |
| safeArea | `Boolean` | SafeArea를 적용할지 여부를 지정합니다. |
| gradationColor | `Color?` | 배경 그라데이션 색상을 설정합니다. |
| variant | `@Composable (() -> Unit)?` | 추가적으로 표시할 컴포넌트입니다. |

## WantedActionArea (Slot을 활용한 커스텀 버튼 전달 방식)
Slot을 활용하여 커스텀 버튼을 직접 전달하는 방식의 액션 영역입니다. 버튼 스타일, 배치 등을 완전히 제어할 수 있습니다.

### 사용 예시
```kotlin
WantedActionArea(
    type = ActionAreaType.Strong,
    positive = {
        CustomMainButton(onClick = { ... })
    },
    negative = {
        CustomSecondaryButton(onClick = { ... })
    }
)
```

### 파라미터
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| type | `ActionAreaType` | 액션 영역의 타입을 설정합니다. |
| safeArea | `Boolean` | SafeArea를 적용할지 여부를 지정합니다. |
| background | `Boolean` | 배경 그라데이션 표시 여부를 지정합니다. |
| gradationColor | `Color?` | 배경 그라데이션 색상을 설정합니다. |
| caption | `String?` | 액션 영역 상단에 표시할 캡션입니다. |
| scrollableState | `ScrollableState?` | 스크롤 가능한 경우 상태를 전달합니다. |
| modifier | `Modifier` | Modifier를 설정합니다. |
| positive | `@Composable (() -> Unit)?` | 메인(긍정) 액션 버튼 Slot입니다. |
| negative | `@Composable (() -> Unit)?` | 서브(부정) 액션 버튼 Slot입니다. |
| neutral | `@Composable (() -> Unit)?` | 추가(중립) 액션 버튼 Slot입니다. |
| variant | `@Composable (() -> Unit)?` | 추가적으로 표시할 컴포넌트입니다. |

## Warning
Deprecated - 더 이상 사용되지 않음. 대신 Slot 방식을 사용하는 WantedActionArea를 사용하세요.

```kotlin
WantedActionArea(
    positive = "확인",
    onClickPositive = { ... },
    actionAreaDefault = WantedActionAreaDefaults.getDefault()
```

## Enum 설명
### ActionAreaType
| 값 | 설명 |
|:---|:---|
| Strong | 강한 강조의 액션 영역 타입 |
| Neutral | 중립적인 액션 영역 타입 |
| Cancel | 취소 타입의 액션 영역 |

---

## Note
WantedActionAreaDefault는 ActionArea에 필요한 버튼 기본 스타일을 묶어둔 데이터 클래스입니다. 각 버튼의 WantedButtonDefault를 개별 설정할 수 있습니다.
