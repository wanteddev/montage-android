## WantedAvatarGroup
여러 개의 아바타를 그룹 형태로 겹쳐 보여주는 컴포넌트입니다.
좌우로 겹쳐진 형태의 아바타와 우측에 추가 텍스트나 콘텐츠를 표시할 수 있습니다.
Drawable 리소스 또는 URL 기반 이미지 모두를 지원합니다.

```kotlin
WantedAvatarGroup(
    modelList = listOf(R.drawable.ic_avatar_placeholder_person),
    modifier = Modifier,
    placeHolder = R.drawable.ic_avatar_placeholder_person,
    size = WantedAvatarSize.Medium,
    type = WantedAvatarType.Person,
    isDrawableRes = true,
)
```

### Parameters
| 이름 | 타입 | 설명 |
|:---|:---|:---|
| modelList | List<Any> | 표시할 아바타 모델 리스트입니다 (URL 또는 Drawable ID). |
| size | WantedAvatarSize | 아바타 크기 및 코너 반경입니다. |
| type | WantedAvatarType | 아바타의 유형(Person, Company, Academic)을 지정합니다. |
| modifier | Modifier | 외형 및 배치를 조정하는 Modifier입니다. |
| placeHolder | Int? | 이미지 로딩 실패 시 사용할 Drawable 리소스 ID입니다. |
| isDrawableRes | Boolean | modelList 항목이 Drawable 리소스인지 여부입니다. |
| trailingContent | (@Composable (Dp) -> Unit)? | 아바타 그룹 오른쪽에 추가적으로 표시할 콘텐츠입니다. |
