## WantedSegmentedControlOutlined

`WantedSegmentedControlOutlined`는 항목을 수평으로 나열하여 하나의 항목을 선택할 수 있도록 하는 UI 컴포넌트입니다.  
선택된 항목은 강조 스타일로 표시되며, 선택 상태에 따라 배경 및 텍스트 색상이 변경됩니다.  
`title` 기반 리스트 방식과 커스텀 아이템 슬롯 방식을 모두 지원합니다.

---



## 텍스트 리스트 기반 사용

### 사용 예시
```kotlin
val items = listOf("One", "Two", "Three")
var selectedIndex by remember { mutableIntStateOf(0) }

WantedSegmentedControlOutlined(
    items = items,
    selectedIndex = selectedIndex,
    onClick = { selectedIndex = it }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| items | `List<String>` | 항목에 표시할 텍스트 리스트입니다. |
| selectedIndex | `Int` | 선택된 항목의 인덱스입니다. |
| modifier | `Modifier` | 외형을 설정하는 Modifier입니다. |
| size | `SegmentedSize` | 항목의 사이즈입니다. Small / Medium / Large 중 선택 가능합니다. |
| onClick | `(Int) -> Unit` | 항목 클릭 시 선택된 인덱스를 반환하는 콜백입니다. |

---



## 슬롯 기반 사용

### 사용 예시
```kotlin
WantedSegmentedControlOutlined(
    itemCount = 3,
    item = { index ->
        WantedSegmentedControlOutlinedItem(
            title = "Item $index",
            isSelected = selectedIndex == index
        )
    },
    onClick = { index -> selectedIndex = index }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| itemCount | `Int` | 항목의 개수입니다. |
| item | `(Int) -> Unit` | 인덱스에 해당하는 항목을 렌더링하는 Composable 슬롯입니다. |
| modifier | `Modifier` | 외형을 설정하는 Modifier입니다. |
| size | `SegmentedSize` | 항목의 사이즈입니다. Small / Medium / Large 중 선택 가능합니다. |
| onClick | `(Int) -> Unit` | 항목 클릭 시 선택된 인덱스를 반환하는 콜백입니다. |

## WantedSegmentedControlOutlinedItem

개별 항목을 구성하는 아이템 컴포넌트로, 선택 상태, 모서리 위치, 아이콘 유무에 따라 스타일이 달라집니다.  
선택 여부에 따라 배경색, 테두리 색, 텍스트 색상이 애니메이션 처리되며, 왼쪽에 아이콘을 배치할 수도 있습니다.

### 사용 예시
```kotlin
WantedSegmentedControlOutlinedItem(
    title = "Option",
    isSelected = true,
    isFirst = true,
    isLast = false,
    icon = {
        Icon(
            painter = painterResource(id = R.drawable.ic_info),
            contentDescription = null
        )
    }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| title | `String` | 항목에 표시할 텍스트입니다. |
| isSelected | `Boolean` | 항목이 선택된 상태인지 여부입니다. |
| modifier | `Modifier` | 외형을 설정하는 Modifier입니다. |
| isFirst | `Boolean` | 첫 번째 항목 여부입니다. 좌측 모서리 라운딩 처리에 사용됩니다. |
| isLast | `Boolean` | 마지막 항목 여부입니다. 우측 모서리 라운딩 처리에 사용됩니다. |
| icon | `(() -> Unit)?` | 항목 왼쪽에 표시할 아이콘 슬롯입니다. 선택 사항입니다. |

---



## Note
- 선택된 항목은 `primary_normal` 색상으로 강조되며, 테두리와 배경 색상이 애니메이션 처리됩니다.
- `WantedSegmentedControlOutlinedItem`을 활용하면 커스터마이징된 아이템 표현이 가능합니다.
