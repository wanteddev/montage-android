## WantedInput

`WantedInput`은 체크박스, 라디오 버튼, 체크마크 형태의 입력 항목을 표시하는 Compose 컴포저블입니다.  
사용자는 다양한 입력 형태를 선택할 수 있으며, 텍스트와 체크 상태를 함께 보여주는 UI를 구성할 수 있습니다.  
내부적으로 `WantedCheckBox` 및 `WantedInputLayout`을 활용하여 UI를 구성하며, 클릭 시 자동으로 상태가 토글됩니다.

### 사용 예시
```kotlin
WantedInput(
    text = "이용약관에 동의합니다.",
    type = WantedInputType.CheckBox,
    size = WantedInputSize.Medium,
    checkBoxState = CheckBoxState.Checked,
    onCheckedChange = { /* 상태 변경 처리 */ }
)
```

### 파라미터

| 이름 | 타입 | 설명 |
|:---|:---|:---|
| text | `String` | 항목에 표시될 텍스트입니다. |
| modifier | `Modifier` | 외형 및 배치를 제어하는 Modifier입니다. |
| type | `WantedInputType` | 입력 타입 (CheckBox, Radio, CheckMark) 중 하나입니다. |
| size | `WantedInputSize` | 컴포넌트 크기 (Medium 또는 Small)입니다. |
| checkBoxState | `CheckBoxState` | 체크 상태 (Unchecked, Checked, Indeterminate)를 설정합니다. |
| bold | `Boolean` | 텍스트를 굵게 표시할지 여부를 설정합니다. |
| enabled | `Boolean` | 항목의 활성화 여부입니다. |
| tight | `Boolean` | true인 경우 컴팩트한 스타일로 표시됩니다. |
| textStyle | `TextStyle` | 텍스트 스타일을 수동으로 지정할 수 있습니다. 지정하지 않으면 size 및 bold에 따라 자동 설정됩니다. |
| interactionSource | `MutableInteractionSource` | 사용자 상호작용 상태를 관리하는 객체입니다. |
| onCheckedChange | `(Boolean) -> Unit` | 체크 상태 변경 시 호출되는 콜백입니다. |

---



## Enum 설명

### WantedInputType
| 값 | 설명 |
|:---|:---|
| CheckBox | 사각형 체크박스입니다. |
| Radio | 원형 라디오 버튼입니다. |
| CheckMark | 체크 아이콘만 표시되는 간단한 스타일입니다. |

### WantedInputSize
| 값 | 설명 |
|:---|:---|
| Medium | 일반적인 높이의 입력 항목입니다. |
| Small | 컴팩트한 높이의 입력 항목입니다. |

---



## Note
- `interactionSource`를 통해 Ripple 효과 및 클릭 이벤트를 조절할 수 있습니다.
- `textStyle` 파라미터로 외부 스타일을 직접 지정하지 않으면, 내부적으로 size와 bold를 기준으로 자동 설정됩니다.
